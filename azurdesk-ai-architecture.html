AzurDesk AI Architecture

Multi-agent helpdesk with ML classification, legal module, mesh discovery, capacity planning and Ollama Cloud integration

Core Backend

  • • Node.js 24 ESM with single HTTP router
  • • better-sqlite3 for local SQLite storage
  • • Zod validation on critical POST endpoints
  • • JWT auth with bcrypt password hashing

AI Agents

  • • ML ticket classifier with compromise/nlp
  • • Agent Rebalance AI for load distribution
  • • Agent Mesh Discovery for expert escalation
  • • Capacity Planner for demand forecasting

Data Layer

  • • Tickets, agents, users, documents
  • • Legal cases, tasks, notes, approvals
  • • Mesh nodes, automaton rules/runs/outbox
  • • Memory graph, KB graph, swarm protocol

Integrations

  • • Ollama Cloud proxy for LLM generation
  • • WebSocket for chat, voice and computer use
  • • PDF/DOCX OCR via documentService
  • • Web builder with static site export
